Lexicons & Resources

A working set of Semitic lexicons and special collections. These large reference works remain on Dr. McDaniel’s original Palmer Seminary website; each link opens there in full.

About the original site. Dr. McDaniel’s original website is hosted by Palmer Theological Seminary at tmcdaniel.palmerseminary.edu. It was last updated on 3 January 2014. Its “Internet Links for Clergy” directory dates largely to the 1990s and early 2000s, and most of those external links no longer work in 2026 — so that directory was not carried over to this archive. Everything Dr. McDaniel wrote is preserved here.
